# 🏦 ECOBANK Credit Scoring AI

Machine learning application for predicting credit risk and supporting financial decision-making.

---

## 📌 Project Overview
This project is an AI-powered credit scoring system designed to assess the risk level of loan applicants.  
It combines a machine learning model with an interactive web interface to provide real-time predictions.

---

## 🚀 Features
- ✅ Credit risk prediction (Low / High risk)
- 📊 Interactive Streamlit dashboard
- 📥 Excel export of results
- ⚡ FastAPI backend for API access
- 🔒 No personal data stored (demo application)

---

## 🛠 Tech Stack
- **Python**
- **Streamlit** (Frontend)
- **FastAPI** (Backend API)
- **Scikit-learn** (Machine Learning)
- **Pandas / NumPy**
